# Calendrier des événements

Cette page affiche un calendrier mensuel dynamique (composant « Powered by JEM ») des événements du Cercle Nautique de France, avec navigation par mois et options « Afficher tout » / « Masquer tout ».

> NOTE DE MIGRATION : Le calendrier est un composant dynamique. Aucun événement spécifique n'était listé (avec date/intitulé) au moment du scraping ; la grille du mois affiché (juillet 2026) apparaissait vide. Les événements sont chargés dynamiquement et devront être récupérés/gérés dans le nouveau site.
